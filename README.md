- [Apache James Server jpa (3.8.1)](https://hub.docker.com/layers/apache/james/jpa-3.8.1/images/sha256-a057ab6b405f2e34b6563502701d164c7b44fb603a4762e21bec7ced2bf4ba59?context=explore) 버전으로 업그레이드

- 사용자 인증 및 보안 기능 강화
